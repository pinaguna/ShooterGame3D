# Unity 3D Shooter Game

A 3D shooter game developed with Unity, featuring basic combat mechanics and interactive level elements.
Players can use a weapon to defeat enemies and progress through the level by unlocking doors.
This project was created to practice core Unity concepts such as 3D movement, shooting mechanics, enemy interactions, and level design.

## Features

Weapon-based shooting mechanics
Defeatable enemy characters
Key collection system to unlock doors
Moving platforms for dynamic level design
3D environment and gameplay
